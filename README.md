чтобы приложение не сломалось на маленьких экранах (iPhone SE vs Pro Max)?»

В ходе работы я столкнулся с тем, что современные ИИ-ассистенты достаточно хорошо 
понимают базовые принципы мобильной верстки, но иногда допускают типичные ошибки, 
связанные с недостаточным учётом платформенных особенностей. А также отлично помогают в дебаге.

Как я контролировал итоговое качество:
1 - Запускал приложение на эмуляторах с разными размерами экрана (small phone, medium phone)
2 - Проверял кликабельность всех элементов и отсутствие перекрытий
